# 마켓플레이스 개요

## 비전

WordPress나 VSCode 확장 생태계처럼, 누구나 모듈을 개발하고 공유할 수 있는 분산화된 마켓플레이스를 구축합니다.

## 핵심 원칙

### 1. 분산화
- 중앙 서버 없음
- GitHub을 통한 배포
- 공식 레지스트리는 검증된 목록만 제공

### 2. 투명성
- 모든 모듈 소스 공개
- 다운로드 통계 공개
- 커뮤니티 리뷰 시스템

### 3. 보안
- 코드 리뷰 필수
- 악성 코드 스캔
- 권한 시스템

### 4. 자유로움
- 누구나 모듈 개발 가능
- 레지스트리 등록 선택 사항
- 직접 설치도 가능

## 구성 요소

```
마켓플레이스 생태계
├── 공식 웹사이트
│   ├── 모듈 검색 및 탐색
│   ├── 통계 대시보드
│   └── 개발자 문서
├── 공식 레지스트리 (GitHub)
│   ├── modules.json (인증된 모듈 목록)
│   ├── 개별 모듈 상세 정보
│   └── 다운로드 통계
├── 앱 내 모듈 관리
│   ├── 설치된 모듈 관리
│   ├── 마켓플레이스 검색
│   └── 원클릭 설치/제거
└── 개발자 도구
    ├── 모듈 템플릿
    ├── 개발 가이드
    └── 제출 프로세스
```

## 앱 내 모듈 관리 페이지

사용자가 UI를 통해 모듈을 관리할 수 있는 대시보드를 제공합니다.

  lastUpdated: Date;

## 개발자 경험

### 모듈 개발 플로우

```
1. 아이디어 구상
   ↓
2. 모듈 템플릿 복사
   ↓
3. 로컬 개발 및 테스트
   ↓
4. GitHub에 업로드
   ↓
5. 공식 레지스트리에 PR 제출
   ↓
6. 코드 리뷰 및 검증
   ↓
7. 승인 → 마켓플레이스에 표시
   ↓
8. 사용자가 설치
   ↓
9. 피드백 및 업데이트
```

### 지원 도구

#### CLI 도구 (향후)
```bash
# 새 모듈 생성
npx create-finance-module my-module

# 로컬 테스트
npm run dev:module my-module

# 배포 준비
npm run build:module my-module

# 레지스트리에 제출
npm run submit:module my-module
```

## 수익화 없음

**중요:** 이 마켓플레이스는 비영리입니다.

- ❌ 유료 모듈 없음
- ❌ 프리미엄 등록 없음
- ❌ 광고 없음
- ❌ 수수료 없음

모든 모듈은 무료이며, 개발자는 자발적 후원만 받을 수 있습니다.
