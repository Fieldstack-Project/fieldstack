# 후원 정책

## 핵심 원칙

### 완전 선택적
- 후원은 **완전히 자발적**입니다
- 후원 여부와 관계없이 **모든 기능 동일**
- 프리미엄 기능 **절대 없음**
- 기능 제한 **절대 없음**

### 투명한 운영
- 수입/지출 **월간 보고서 공개**
- 사용처 **투명하게 공개**
- 커뮤니티와 함께 **의사결정**

## FAQ

### Q1. 후원하지 않으면 기능이 제한되나요?
**A:** 아니요. 모든 기능은 후원 여부와 관계없이 동일합니다.

### Q2. 후원금은 어디에 사용되나요?
**A:** 인프라 비용, 개발 도구, 개발자 보상 등에 사용되며, 매월 투명하게 공개됩니다.

### Q3. 회사에서 후원하려면?
**A:** sponsors@your-finance-system.dev로 문의해주세요.

### Q4. 익명으로 후원할 수 있나요?
**A:** 네, 익명 후원이 가능합니다.

### Q5. 환불 가능한가요?
**A:** 월 단위 후원은 환불이 불가능합니다. 언제든 취소는 가능합니다.

### Q6. 세금 공제 가능한가요?
**A:** 이 프로젝트는 비영리 단체가 아니므로 세금 공제 대상이 아닙니다.
